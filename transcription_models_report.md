# Отчет о сравнении моделей транскрипции

## Введение

Данный отчет представляет собой сравнительное исследование трех моделей транскрипции аудио от OpenAI:
1. **whisper-1** (традиционная модель Whisper)
2. **gpt-4o-transcribe** (новая модель на базе GPT-4o, оптимизированная для точности)
3. **gpt-4o-mini-transcribe** (компактная версия, оптимизированная для скорости)

Цель исследования - определить оптимальные модели для различных сценариев использования, с особым фокусом на русскоязычных и англоязычных аудиофайлах разной длительности.

## Методология тестирования

Для проведения тестирования были использованы следующие типы аудиофайлов:
- Короткие русские аудиофайлы (5-10 секунд)
- Средние русские аудиофайлы (30-60 секунд)
- Длинные русские аудиофайлы (более 2 минут)
- Короткие английские аудиофайлы (5-10 секунд)
- Средние английские аудиофайлы (30-60 секунд)
- Длинные английские аудиофайлы (более 2 минут)

Для каждого типа аудиофайлов были измерены следующие параметры:
- Скорость транскрипции (в секундах)
- Точность транскрипции (качественная оценка)
- Стабильность результатов (вариативность между попытками)

## Общие выводы и рекомендации

### Сравнение моделей

| Аспект | whisper-1 | gpt-4o-transcribe | gpt-4o-mini-transcribe |
|--------|-----------|-------------------|------------------------|
| Скорость | Высокая для длинных файлов | Низкая | Средняя |
| Точность (рус) | Средняя | Высокая | Хорошая |
| Точность (англ) | Высокая | Высокая | Хорошая |
| Стабильность | Высокая | Высокая | Средняя |
| Размер модели | Средний | Большой | Малый |

### Рекомендации по выбору модели

1. **Для русского языка:**
   - Короткие аудио (до 30 сек): **gpt-4o-mini-transcribe**
   - Средние аудио (30 сек - 5 мин): **gpt-4o-transcribe**
   - Длинные аудио (более 5 мин): **whisper-1**

2. **Для английского языка:**
   - Любая длительность: **whisper-1**
   - Сложные акценты или терминология: **gpt-4o-transcribe**

3. **При ограниченных вычислительных ресурсах:**
   - Для всех языков: **whisper-1** или **gpt-4o-mini-transcribe**

4. **При высоких требованиях к точности:**
   - Для русского: **gpt-4o-transcribe**
   - Для английского: **whisper-1** или **gpt-4o-transcribe**

### Особенности использования моделей

1. **Важность указания языка:**
   - Для всех моделей указание языка значительно повышает точность транскрипции
   - Наибольший эффект наблюдается для **whisper-1** и **gpt-4o-mini-transcribe**

2. **Использование подсказок:**
   - Подсказки особенно эффективны для моделей **gpt-4o-transcribe** и **gpt-4o-mini-transcribe**
   - Для модели **whisper-1** эффект менее выражен

3. **Обработка шумов:**
   - Лучшую устойчивость к шумам демонстрирует **gpt-4o-transcribe**
   - **whisper-1** более чувствительна к качеству аудио

4. **Оптимизация аудиофайлов:**
   - Конвертация в формат MP3 с параметрами 16kHz, 32kbps, mono дает наилучшие результаты для всех моделей
   - Такая предварительная обработка улучшает результаты и снижает размер файлов

## Алгоритм автоматического выбора модели

На основе результатов тестирования был разработан алгоритм автоматического выбора оптимальной модели транскрипции для конкретного аудиофайла:

1. **Определение языка:**
   - Если язык не указан явно, используется кратковременная предварительная транскрипция с **whisper-1** для определения языка

2. **Анализ длительности файла:**
   - Короткие файлы (до 30 сек)
   - Средние файлы (30 сек - 5 мин)
   - Длинные файлы (более 5 мин)

3. **Выбор модели на основе языка и длительности:**
   - Для русского языка:
     - Короткие: **gpt-4o-mini-transcribe**
     - Средние: **gpt-4o-transcribe**
     - Длинные: **whisper-1**
   - Для английского языка:
     - Любая длительность: **whisper-1**
   - Для других языков:
     - Короткие: **gpt-4o-mini-transcribe**
     - Средние и длинные: **whisper-1**

4. **Учет пользовательских предпочтений:**
   - Если указано предпочтение скорости, выбирается более быстрая модель
   - Если указано предпочтение точности, выбирается более точная модель

## Заключение

Проведенное тестирование показало, что для русскоязычных аудиофайлов модели на базе GPT-4o обеспечивают более высокую точность транскрипции по сравнению с традиционной моделью whisper-1. Однако для англоязычных аудиофайлов преимущества новых моделей менее выражены, и whisper-1 остается оптимальным выбором с точки зрения соотношения скорости и качества.

Автоматический выбор модели на основе языка, длительности файла и пользовательских предпочтений позволяет достичь оптимального баланса между скоростью и качеством транскрипции для различных типов аудиофайлов.
