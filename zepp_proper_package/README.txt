## Приложение для записи аудио на Zepp OS

Это приложение позволяет записывать аудио на умных часах с Zepp OS и отправлять его на сервер.

### Особенности
- Запись аудио с микрофона часов
- Фрагментирование длинных записей
- Автоматическая отправка на сервер
- Повторные попытки при сбоях сети
- Сохранение записей на часах при отсутствии интернета

### Совместимые устройства
- Amazfit GTR 3 Pro
- Другие устройства с Zepp OS 3.0+

### Настройка сервера
Для работы с приложением необходим сервер с настроенными API эндпоинтами.
Подробная информация в файле server_config.md
