# Инструкция по установке и настройке симулятора Zepp OS

Для проверки работы приложения без установки на физическое устройство, можно использовать официальный симулятор Zepp OS. Вот как его установить и настроить:

## Шаг 1: Скачивание симулятора

1. Скачайте Zepp OS Simulator Beta с официального сайта Zepp:
   - [Zepp OS Simulator Beta v0.1.0](https://docs.zepp.com/docs/guides/tools/simulator/download/)
   - Прямая ссылка: https://resource.zepp.com/zcospub/Zepp-OS-Simulator-Beta-v0.1.0.zip

## Шаг 2: Установка симулятора

1. Распакуйте скачанный ZIP-архив
2. Для Windows: Запустите `Zepp-OS-Simulator.exe`
3. Для macOS: Откройте `Zepp-OS-Simulator.app`
4. Для Linux: Запустите соответствующий исполняемый файл

## Шаг 3: Настройка симулятора для нашего приложения

1. Скачайте подготовленный пакет приложения `zepp_app_bundle.zab` из данного репозитория
2. В интерфейсе симулятора нажмите кнопку "Open App"
3. Выберите скачанный файл `zepp_app_bundle.zab`
4. Выберите модель устройства GTR 3 Pro из выпадающего списка

## Шаг 4: Настройка подключения к серверу

По умолчанию приложение настроено на подключение к локальному серверу по адресу:
```
http://localhost:5000
```

Если вы хотите подключиться к другому серверу:
1. Откройте файл `app/utils/api.js` внутри симулятора (если эта функция поддерживается)
2. Или измените файл в пакете и переупакуйте его перед загрузкой в симулятор

## Шаг 5: Тестирование функциональности

1. Запустите симулятор с загруженным приложением
2. Убедитесь, что ваш сервер запущен и доступен
3. В симуляторе:
   - Откройте приложение "AudioRecorder"
   - Нажмите кнопку "Начать запись"
   - Запись будет симулироваться в течение 30 секунд
   - Нажмите "Остановить запись"
   - Проверьте логи отправки фрагментов на сервер

## Известные ограничения симулятора

1. Симулятор может не поддерживать реальную запись звука - он имитирует процесс
2. Некоторые функции API Zepp OS могут работать иначе, чем на реальном устройстве
3. Доступ к файловой системе и использование датчиков ограничены

## Альтернативный подход: Тестирование без симулятора

Если симулятор недоступен, вы можете использовать тестовый скрипт:
```bash
node test-zepp-integration.js
```

Этот скрипт эмулирует процесс записи и отправки аудио с устройства Zepp OS.
