2. **Пример кода для взаимодействия с `gpt-4o` в реальном времени с использованием Python**:
   - Этот пример демонстрирует, как использовать API `gpt-4o` для обработки аудио в реальном времени:
     ```python
     import openai
     import speech_recognition as sr

     openai.api_key = "YOUR_API_KEY"

     # Инициализация распознавателя речи
     recognizer = sr.Recognizer()

     # Бесконечный цикл для обработки аудио ввода
     while True:
         try:
             # Прослушивание аудио ввода
             with sr.Microphone() as source:
                 print("Listening...")
                 audio = recognizer.listen(source)

             # Транскрипция аудио с использованием GPT-4o
             transcription = openai.Audio.transcribe(
                 model="whisper-1",
                 file=audio,
             )

             print("Transcription:", transcription)
         except Exception as e:
             print("Error:", e)
     ```
