Скачать приложение для Zepp OS

Это приложение для умных часов с Zepp OS позволяет записывать аудио и отправлять на сервер.

Скачать приложение (.zab)

Инструкция по установке

Шаг 1: Скачать файл

Нажмите кнопку "Скачать приложение" выше и сохраните файл на ваше устройство.

Шаг 2: Перенести файл на смартфон

Передайте файл zepp_app_bundle.zab на ваш смартфон.

Шаг 3: Установить через приложение Zepp

Откройте приложение Zepp на смартфоне и перейдите в раздел "Профиль" -> "Мое устройство" -> выберите ваши часы -> "Установить приложение" и укажите путь к файлу .zab
